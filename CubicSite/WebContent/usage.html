CubicTest quick start

Create a New CubicTest Project by right-clicking in the navigator view:
  • New -> New CubicTest Project   (follow the instructions on screen)

An empty test will appear in the Editor window. You can now add page elements (e.g. assert text present) to the page that appears.

To create a user interaction that leads from one page to another:

  • Right click on page -> Add user interaction

To run the test:

  • Right click in the test editor -> Run Test with Cubic Selenium Runner
