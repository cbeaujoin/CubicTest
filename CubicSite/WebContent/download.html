Update site

CubicTest can also be installed into an existing Eclipse installation throught the Eclipse Update Manager.
The URL is: http://www.cubictest.org/update
Installation details
Minimum requirements for the update site-version of CubicTest:
Eclipse 3.3, Eclipse GEF 3.3 and Java Runtime Environment 5.0. If you encounter problems, see the troubleshooting page

SVN
