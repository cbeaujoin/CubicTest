Update Site

CubicTest Eclipse Plug-in (platform independent)
Requires minimum Eclise 3.3 with Java 5 JRE.
  1. First install Eclipse GEF (required by CubicTest):
    Update Manager URL:  http://download.eclipse.org/tools/gef/update-site/releases
  2. Then install CubicTest:
    Update Manager URL:  http://www.cubictest.org/update
Detailed installation instructions

If you are upgrading from a previous version, you need to uninstall CubicTest first.
In Eclipse: Help -> Software Updates -> Manage Configuration -> Expand Eclipse platform features and locate CubicTest -> Right Click and select "Uninstall".

If you encounter problems, see the troubleshooting page

Rich client download

A rich client version of CubicTest with bundled Java 5 JRE and Eclipse can be downloaded here:
CubicTest Rich Client downloads (win32 only)

It is recommended for users without Eclipse and Java, and requires no installation except downloading and extracting a zip file.

Users of other OS'es than Windows must use the Update Site version.
