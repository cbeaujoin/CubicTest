Features

CubicTest

  • is an open source Eclipse plugin
  • uses a graphical user interface to let users model tests instead of writing test scripts
  • focuses on both testing existing applications and test driven development of new web applications
  • make tests easier to design, understand and run for nontechnical as well as technical users
  • exports to executable Selenium and Watir tests

Unique features in CubicTest

  • Tests can be run directly in the Eclipse GUI for fast feedback
  • Visual test composition and test orchestration to make it easier to read and write larger test sets. Sub-tests can be used to create a domain language in the test
  • A way to intuitively and robustly identify elements on a web page by looking at neighbouring or surrounding elements ("contexts")
  • Exporter architechture for easy additions of exporters/runners of the test model
  • A recorder for recording tests from existing web applications (based on Selenium RC)
  • A sample exporter for creating a mock-up (HTML skeleton) of the modeled web application based on the tests, with e.g. forms and site navigation present

Automated running of tests with Maven 2

  • CubicTest tests can be run standalone by a Maven 2 build without starting or depending on Eclipse.
