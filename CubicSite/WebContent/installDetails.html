Installing CubicTest

CubicTest requires Java J2SE 5.0 and Eclipse 3.1 (or higher).

Links to external download pages:
  • Java     NB:  J2SE 5.0 is required!
  • Eclipse 3.1

When Java and Eclipse is installed, use the "Eclipse software updates" feature to install CubicTest into Eclipse.

In Eclipse, go to Help -> Software updates -> Find and install

Select "Search for new features to install":

First install the Eclipse Graphical Editing Framework (GEF) by using the standard Eclipse.org update site:

Press "Finish" in the dialog above and select a mirror near you.

This will launch the wizard for installing GEF. Select to install the Graphical Editing Framework 3.1.0 (or higher) and press the "Next" button.

Complete the install wizard that opens. Restart Eclipse if you are prompted to do so.

If you are having problems with installing GEF, try to download it manually from http://www.eclipse.org/gef/ and unzip it into your eclipse plugins directory.

When GEF is installed, continue with the installation of CubicTest:

As before, the following window should be active: Eclipse -> Software updates -> Find and install -> Search for new features to install.

Select "New Remote Site..."

Enter URL:http://cubictest.sourceforge.net/update/

Make sure only CubicTest is selected, and press "Finish":

Complete the install wizard that opens. Restart Eclipse if you are prompted to do so.

Your're done installing CubicTest!
