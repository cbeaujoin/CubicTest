Upgrading the Update Site version of CubicTest

If you are upgrading from a previous version, use the standard Eclipse plugin upgrade mechanism:
  • In Eclipse select Help -> Software Updates -> Manage Configuration. Locate CubicTest (expand Eclipse), right click on it and select "Find Updates...". Complete the standard Eclipse plugin upgrade process.
  • When Eclipse is restarted, you need to Reset the Perspective to get shortcuts to the new features . In Eclipse, select Window -> Reset Perspective.
  • You should also edit your projects pom.xml file and upgrade to the 1.8.6 of the cubictest maven 2 plugin referenced there.
